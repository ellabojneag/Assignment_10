How does weather change with latitude and longitude?


The purpose of this website is to analyze how weather changes as it approaches the equator. To accomplish this analysis, we pulled data from an OpenWeatherMap API to assemble a dataset of over 500 cities.

After assembling the dataset, we used matplotlib to plot how various attributes of weather change with latitude, such as temperature, cloudiness, and humidity. This site provides the source data and visualizations created as part of an analysis, as well as attempts to explain correlations and describe trends in the visualization.
